# Roman to integer

문제는 간단하다. 로마 숫자를 숫자로 변환하면 된다. ( 로마식 표기법을 변환 )  
근데 조건이 상당히 더럽다.
별로 좋은 문제는 아닌 것 같다.



~~~
Example 1:

Input: s = "III"
Output: 3
Explanation: III = 3.

Example 2:

Input: s = "LVIII"
Output: 58
Explanation: L = 50, V= 5, III = 3.

Example 3:

Input: s = "MCMXCIV"
Output: 1994
Explanation: M = 1000, CM = 900, XC = 90 and IV = 4.
~~~
